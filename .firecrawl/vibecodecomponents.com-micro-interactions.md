Copy

```
Create animated number counters:
- Counters start at 0 and animate to target value
- Use requestAnimationFrame for smooth animation
- Apply easing function (ease-out cubic) for deceleration
- Only trigger animation when element enters viewport
- Format large numbers (e.g., 12500 → 12.5K)
- Include replay button to reset counters
- 2 second animation duration.
```

# Prompt Library — Category Index

The library is split into ten category files. When generating a site, load **only** the categories relevant to the request (see SKILL.md Step 3) — not this whole directory.

| Category file | What it holds | Typically used for tiers |
|---|---|---|
| `hero-sections.md` | Above-the-fold hero layouts, headlines, CTAs | 1-5 |
| `scroll-effects.md` | Scroll-triggered reveals, parallax, pinned sections | 2-5 |
| `3d-components.md` | WebGL / Three.js / Spline embeds, 3D objects | 4-5 |
| `interactive-elements.md` | Hover states, cursors, toggles, micro-interactions | 2-5 |
| `navigation.md` | Navbars, menus, mobile nav, sidebars | 1-5 |
| `cards-and-grids.md` | Card layouts, bento grids, feature/pricing grids | 1-5 |
| `forms-and-inputs.md` | Contact forms, inputs, multi-step forms, auth | 1-4 |
| `footers.md` | Footer layouts, link columns, newsletter signups | 1-5 |
| `animations.md` | Entrance animations, transitions, loaders, text effects | 2-5 |
| `backgrounds.md` | Gradients, patterns, noise, animated/generative backgrounds | 1-5 |

Each file is empty until the harvest phase populates it. See `harvest/HARVEST_GUIDE.md`.

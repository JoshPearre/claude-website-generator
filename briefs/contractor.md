## Niche Design Brief — contractor / home services

- Niche & read: lead-gen site — competent, insured, local; must read "shows up on time", not "startup"
- Tier: 3 — Modern Consumer (local-business default; conservative trades can sit at 2)
- Design direction: modern-minimal + accent override
- Palette mood: high-trust neutrals + one work-wear accent, strong contrast, light default
  → --accent ≈ oklch(60% 0.15 60) safety amber (or steel blue oklch(55% 0.12 250) for plumbing/HVAC)
- Type character: bold, condensed-leaning sans display, heavy weights, all-caps labels; plain readable body
- Imagery direction: real jobsite photography — crews at work, before/after pairs, honest daylight; no stock-suit handshakes
- Image query seeds: ["construction crew residential jobsite daylight", "home renovation before after interior", "contractor tools workbench detail"]
- Layout & density: medium density; hero/services/before-after/service-area+badges/quote-form/footer
- Motion character: minimal — hover states + one count-up stat strip; registry: none
- Shape/texture: small radii (4–8px), visible 1px borders, badge/stamp motifs for licenses & insurance
- Native components: hero=photo + inline quote form · nav=phone number top-right, always visible · card=service tiles with icon
- ui-ux-pro-max: style=sturdy utilitarian modern · UX focus=click-to-call, quote-form friction, trust badges
- Prompt categories: hero-sections, navigation, cards-and-grids, forms-and-inputs, footers
- One deliberate departure: an oversized before/after slider as the centerpiece section
- Anti-references: clip-art hard hats, red/yellow/black bumper-sticker palettes, walls of certification logos, generic blue-gradient corporate
- Reference brands: Ergeon, Block Renovation, Angi (pro side) · galleries: Land-book "Services", SiteInspire "Professional Services"

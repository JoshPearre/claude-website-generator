## Niche Design Brief — salon / spa

- Niche & read: bookings site — calm, elegant, aspirational self-care; must read premium without reading expensive-cold
- Tier: 3 — Modern Consumer (tier-niche-map default)
- Design direction: human-approachable + accent override
- Palette mood: soft, desaturated warm neutrals, gentle contrast, light default
  → --accent ≈ oklch(62% 0.09 20) muted rose/clay (keep direction bg/fg)
- Type character: refined sans or low-contrast serif display, light-to-medium weights, generous letter-spacing on labels
- Imagery direction: close-crop detail shots (hands, hair, texture), soft diffused daylight, calm expressions, real photography
- Image query seeds: ["salon hair styling close-up soft natural light", "spa treatment calm neutral tones minimal", "modern salon interior bright airy"]
- Layout & density: airy; hero/services+prices/gallery/team/book/footer — services as a clean price list, not cards-on-cards
- Motion character: light reveals only, slow fades (whileInView, 300–400ms); registry: none
- Shape/texture: soft radii (12–16px), no hard borders — whitespace does the separating
- Native components: hero=split image + book CTA · nav=minimal, book button right · card=service+price rows
- ui-ux-pro-max: style=soft premium minimalism · UX focus=booking-flow clarity, price legibility, mobile-first
- Prompt categories: hero-sections, navigation, cards-and-grids, forms-and-inputs, footers
- One deliberate departure: one full-bleed texture moment (hair/skin macro) as a section divider
- Anti-references: pink-glitter clichés, lash-extension stock collages, cluttered price grids, purple-gradient spa slop
- Reference brands: Aesop, Gee Beauty, Chillhouse · galleries: SiteInspire "Beauty", Land-book, One Page Love

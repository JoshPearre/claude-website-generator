## Niche Design Brief — gym / fitness studio

- Niche & read: membership site — energetic, motivating, community-forward; "energetic but balanced" (tier map), not supplement-ad aggressive
- Tier: 3 — Modern Consumer (tier-niche-map default)
- Design direction: modern-minimal + accent override, dark-leaning
- Palette mood: high contrast, near-dark default with one electric accent
  → --accent ≈ oklch(75% 0.19 130) electric lime (or oklch(60% 0.22 25) signal red for combat/strength gyms)
- Type character: heavy condensed sans display, aggressive weight contrast, tight leading, all-caps section labels
- Imagery direction: mid-effort training shots (not posed flexing), hard directional gym light, sweat/chalk texture, real members over models
- Image query seeds: ["athlete training gym dramatic light sweat", "group fitness class energy candid", "gym interior dark moody equipment"]
- Layout & density: punchy; hero/programs/schedule/coaches/pricing/join/footer
- Motion character: medium — stagger reveals + one count-up stats band; registry: Magic UI (marquee of class names, number ticker) if any
- Shape/texture: sharp radii (0–6px), thick rules, diagonal crops, grain on photos
- Native components: hero=full-bleed action photo + join CTA · nav=bold, trial CTA · card=program cards with schedule chips
- ui-ux-pro-max: style=high-energy dark athletic · UX focus=pricing clarity, class-schedule scanability, mobile join flow
- Prompt categories: hero-sections, navigation, cards-and-grids, animations, forms-and-inputs, footers
- One deliberate departure: an oversized vertical "JOIN" rail or giant outlined display word as a section anchor
- Anti-references: flexing-bicep stock, red/black grunge-brand energy, cluttered supplement-ad layouts, fake countdown timers
- Reference brands: Barry's, Gymshark (editorial pages), Equinox · galleries: Land-book "Sports & Fitness", SiteInspire "Sport"

# Niche → Tier Map

Maps common website niches to a default design intensity tier. The `website-generator` skill reads this in Step 2. Edit freely — add niches, change tiers, adjust notes.

Tiers: **1** Utility · **2** Restrained Professional · **3** Modern Consumer · **4** Expressive Brand · **5** Experimental/Maximalist.

**Ambiguity guidance:** some niches split by purpose. "Restaurant" = Tier 1 if it's an ordering/POS dashboard, Tier 3 if it's a marketing/menu site. "SaaS" = Tier 1 for the in-app dashboard, Tier 2-3 for the marketing site. Always use the user's "what's it about" answer to disambiguate; default to **Tier 3** when truly unclear, and say so.
